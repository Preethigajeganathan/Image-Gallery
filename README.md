# 🖼️ Image Gallery

A modern and responsive **Image Gallery** built using **HTML, CSS, and JavaScript**. The gallery supports horizontal scrolling, responsive layouts for different screen sizes, smooth navigation controls, and interactive hover effects.

## 🚀 Live Demo

🔗 https://preethigajeganathan.github.io/Image-Gallery/

## 🚀 Features

* Responsive image gallery layout
* Horizontal scrolling using mouse wheel
* Previous and Next navigation buttons
* Smooth scrolling animation
* Hover enlargement effect
* Grayscale to color image transition
* Responsive image sizing
* Mobile-friendly design
* Tablet-friendly design
* 4K screen support
* Custom favicon support

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Image-Gallery.git
```

### 2. Open the Project Folder

```bash
cd Image-Gallery
```

### 3. Launch the Application

Open `index.html` in any modern web browser.

No additional installation is required.

## 🎮 Usage

1. Open the gallery in your browser.
2. Scroll horizontally using the mouse wheel.
3. Click the **Next** button to move forward.
4. Click the **Back** button to move backward.
5. Hover over images to reveal interactive effects.
6. Enjoy a responsive gallery experience across all devices.
